# 医疗AI数据分析可视化系统

## 项目简介
基于Vue3 + ECharts的医疗数据分析可视化系统，包含数据表格页面和可视化大屏页面。

## 技术栈
- Vue 3 + Vite
- Element Plus (UI组件库)
- ECharts (数据可视化)
- Axios (HTTP请求)
- MySQL (后端数据库)

## 功能模块
1. **数据表格页面** - 展示病例数据列表，支持分页、筛选、搜索
2. **可视化大屏页面** - 包含多种图表统计：
   - 各年龄段患病占比环形图
   - 疾病类型分布横向条形图
   - 疾病数据信息卡片
   - 男女性别患病对比饼图
   - 医院科室环形图
   - 疾病关键词云图
   - 病例列表
   - 身高体重平均数柱状图

## 启动方式
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
